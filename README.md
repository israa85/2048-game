# 2048 Game

A Python implementation of the popular 2048 puzzle game using Pygame.

1. **Controls:**
   - Arrow keys: Move tiles 
   - UNDO button: Undo last move (available after creating 128 tile)
   - RESET button: Start a new game

2. **Objective:**
   - Combine tiles with the same number
   - Reach the 2048 tile (primary)
   - Try to achieve the highest score possible

## Game Mechanics

- New tiles (2 or 4) spawn after each move
- Tiles merge when matching numbers collide
- Game ends when no moves are possible
- Score increases by the value of merged tiles
